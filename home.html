Welcome to Webtechnology

Internet is the most relevant information technology of our age. World Wide Web is its most important application. In this course we will learn the architecture of WWW, its main protocols, representation formats and development technologies. The practical component of the course will focus on the design and implementation of Web applications. Additional topics such as Semantic Web, Social Web and Adaptive Web will help us better understand current trends of Web development.

Course material is roughly divided into 4 learning units:

  • Client representation (Lectures 1-3),
  • Client programming (Lectures 4-8),
  • Server programming (Lectures 9-13),
  • Trends of Web Development (Lectures 14-16).

Course prerequisites

The course has no formal prerequisites. However, basic knowledge of programming are expected. Although this is an introductory course, many students taking it have had experience with some aspects of Web development. If you do not have any knowledge of HTML, CSS and JavaScript, you definitely can still get a high grade in the course, but you will need to work hard for it.

Working methods

  • Lectures. Most of the course material is delivered through standard lectures accompanied by PowerPoint slides. Lectures take place twice a week. After each lecture, its slides are made available on the course page.
  • Practical classes. Practical classes follow the lectures and are conducted by teaching assistants. They can help you find answers to practical questions and progress with your homework assignments.
  • Homework assignments. Homework assignments are set out to help you practice knowledge acquired during lectures. They are done in groups of 3. You are responsible for forming your group. Homework will gradually develop, thus you will be able to reuse and develop your code throughout the course. Keep in mind, that assignment 3 is much more challenging than the first two. You are given enough time to complete it, make sure to start using this time as soon as possible.
  • Exams. The course has two main exams: the midterm and the final. The midterm covers the material of the first 8 lectures. The final covers the last 8. Both exams are digital; you will take them in the UU assessment system Remindo. Both exams are closed books. You will have 100 minutes to answer exam questions, if you are a student registered for an extra time during exams, you will have 120 minutes. Some exam questions are more practical, and some will verify your knowledge of important concepts and facts.
  • Additional reading. The course does not have a single textbook. Most Web technology textbooks become obsolete rather fast due to constant introduction of new technologies and evolution of exciting ones. Instead, plenty of online resources structured around course topics are made available for you. You are are not required to read all the suggested resources. They are there to help you understand challenging topics and ensure you have enough points of reference to complete homework assignments.

Course Outcomes

This course is designed to help you understand how Web applications operate and learn how to build them yourself.

After completing Unit 1, you will:

  • know the main Internet communication protocols and understand their characteristics;
  • be able to write well-structured, easily maintained, standards-compliant HTML5 code;
  • understand the need and requirements of the accessibility standards and be able to produce accessible HTML content;
  • be able to write well-structured, easily maintained, standards-compliant CSS3 code;
  • understand the need and requirements of the responsive design and be able to tailor the presentation of your HTML to users’ platforms using CSS-based responsive design.

After completing Unit 2, you will:

  • Understand the basics of JavaScript language (variables, basic data types, control flow, functions) and be able to write simple JavaScript code
  • Understand Document Object Model and be able to access and modify it with the help of JavaScript
  • Be able to use JavaScript to implement interactive, event-based Web pages
  • Understand the relations between functions and prototypes, understand the nature of JavaScript “classes”, be able to write OOP code in JavaScript
  • Understand the flexible nature of JavaScript functions, understand the concept of callback functions, be able to write basic functional code with JavaScript
  • Understand such components of the JavaScript browser environment as event loop, call stack and callback queue, be able to write basic asynchronous JavaScript code
  • Be able to use JavaScript frameworks
  • understand the basics of XML and XML Schema, understand JSON, be able to process XML and JSON with JavaScript 5

After completing Unit 3, you will:

  • Understand the typical 3-tier client-server architecture of Web applications
  • Understand the characteristics of Node.JS as the Web server development technology
  • Understand the workflow of a Web server and Node.JS/Express.JS server in particular
  • Be able to implement typical Express middleware (logging, routing, error handling, serving static files, DB access, session management)
  • Understand how to maintain and control user state in Web applications between the client and the server
  • Understand cookies, their characteristics and categories, be able to create, modify, delete, process and communicate cookies
  • Understand the basics of session management in Web applications, be able to create and maintain sessions
  • Understand AJAX-based communication between client and server, be able to program interactive Web-application suing AJAX
  • understand the basic of DB management in Web applications, be able to program communication between an SQLite data base and an Express.JS web server
  • Understand the basics of Web security, recognise main vulnerabilities ofWeb applications according to the OWASP list of top ten
  • Be able to implement secure Web applications

After completing Unit 4, you will:

  • Recognise several trends of Web development (Adaptive Web, Semantic Web, Social Web)
  • Understand the concept of personalisation, its role in Web applications and problems it can help to solve
  • Understand the concept of a user model/profile, how it is populated and used in adaptive applications
  • Understand the task of personalised Web search and how Web search can be personalised
  • Understand the task of adaptive hypermedia and different techniques it uses
  • Understand the task of adaptive recommendation and collaborative filtering approach in particular
  • Understand the potential dangers of Adaptive Web 6
  • Understand the purpose of Semantic Web technologies
  • Understand the basics of RDF, RDFS and OWL
  • Understand the concepts of metadata and linked data
  • Understand the basics of Schema.org
  • Understand the characteristics of Social Web applications
  • Understand the challenges of building Social Web applications
